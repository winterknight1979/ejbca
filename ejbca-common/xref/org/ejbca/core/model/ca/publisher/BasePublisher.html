170      * @return true if the certificate should be published.

174     /**
175      * Publishes a certificate to a certificate store. If status is not active for the certificate, the publisher may choose
176      * to not publish the certificate, for instance if revoke removes a certificate from LDAP,
177      * re-publishing the certificate should not add it again if the status is revoked.
178      *
179      * To revoke a certificate (already revoked by the CA) call with status=CertificateDataBean.CERT_ACTIVE, the Publisher decides what to do, if
180      * anything.
181      * @param admin admin 
182      * 
183      * @param incert The certificate to be stored.
184      * @param username Username of end entity owning the certificate.
185      * @param password Password given to the user, may be null if no password exists for the user.
186      * @param userDN if a DN object is not found in the certificate use object from user data instead, can be null.
187      * @param cafp Fingerprint (hex) of the CAs certificate.
188      * @param status Status of the certificate (from CertificateDataBean.CERT_ACTIVE, CERT_REVOKED etc).
189      * @param type Type of certificate (from CertificateDataBean.CERTTYPE_ENDENTITY etc).
190      * @param revocationDate Date for revocation (of revoked), like System.currentTimeMillis(), or -1 if not revoked.
191      * @param revocationReason reason for revocation from RevokedCertInfo, RevokedCertInfo.NOT_REVOKED if not revoked.
192      * @param tag  TAg
193      * @param certificateProfileId Profile
194      * @param lastUpdate Date
195      * @param extendedinformation contains extended information about the user, like picture, is null if no extendedinformation exists about the user.
196      *
197      * @return true if storage was successful.
198      *
199      * @throws PublisherException if a communication or other error occurs.
200      */    
201     public abstract boolean storeCertificate(AuthenticationToken admin, Certificate incert, String username, String password, String userDN, String cafp, int status, int type, long revocationDate, int revocationReason, String tag, int certificateProfileId, long lastUpdate, ExtendedInformation extendedinformation) throws PublisherException;
