About EJB-CA

"New" EJB CA for Java 11 and modern app servers Forked from PrimeKey EJB-CA CE 6.15.2.6

Project Modules

This project has declared the following modules:

Name Description
CeSeCore Security - Common Code Common code from CeSeCore
EJBCA Shared resource bundle Shared resource bundle; configurator, and Service Manifest builder.
CeSeCore Security JPA Entities Java Persistence API entities for CeSeCore
Unit Test Aggregator This module is used as an aggregator for (a) test coverage and (b) any tests that would create circular deps. It should explicitly depend on all other modules.
CeSeCore Security EJB Interface Public interfaces for CeSeCore EJB's
CeSeCore Security - EJB Implementation Implementation of CeSeCore Enterprise JavaBeans
EJB-CA Common code Common code for all EJB-CA modules
EJBCA JPA Entities Java Persistence API entities and descriptor for EJBCA
EJBCA Webservices - common code Common code for EJBCA webservices. Note: Upstream combines ejbca-ws-common and ejbcs-ws-ejb into a single jar. They've been split here to avoid a circular dependency on ejbca-entity.
EJB-CA EJB Interfaces Public interfaces for EJB-CA EJB's
EJB-CA Common Web code Common code for EJB-CA web interfaces
EJBCA Public web GUI Web GUI frontend for EJBCA.
